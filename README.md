# fraction-part-of-stantard-IEEE-754
This small code implements a partial portion of the IEEE 754 standard to do without calculator.
¿Quieres clonar este repo en tu entorno de codigo?
OKEY TE INDICO LOS PASOS
1.- Crea una carpeta donde vayas a clonar este repositorio / si ya la tienes solo abrela dentro de tu entorno de codigo
2.- Dentro de un entorno visual abre la terminal y escribe git clone https://github.com/jors2007/fraction-part-of-stantard-IEEE-754.git
3.- TADA, ya tienes el codigo en tu carpeta
Nota: Recuerda que la parte fraccionaria tiene que ser de tu numero decimal ya transformado a base 2, suerte. :D
